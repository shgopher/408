# 双指针

基础知识：常见双指针算法分为三类，同向 (即两个指针都相同一个方向移动)，背向 (两个指针从相同或者相邻的位置出发，背向移动直到其中一根指针到达边界为止)，相向 (两个指针从两边出发一起向中间移动直到两个指针相遇)
